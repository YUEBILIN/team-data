# MCP 方案补充：Pivot Web 的「外部 AI 接入」设置页

004 / 005 整体认同。只补充一点被低估的事——**客户端第一次连上 `pivot.enclaws.ai/mcp` 这一步**，如果 Pivot Web 不提供配套引导，非技术用户会卡壳。

## 一、问题

004 §四 说"配一个 URL + PAT 就能用"。技术上对，但对非技术用户隐藏了三件事：

1. 配置文件在哪找（Mac / Win / Linux 路径各异）
2. JSON / TOML 语法改错会让客户端打不开
3. 不同客户端配置格式不同，不能复用同一份内容

## 二、原理：客户端差异决定了不能"一份配置走天下"

| 维度 | Claude Desktop | Claude Code | Codex | Cursor |
|---|---|---|---|---|
| 配置格式 | JSON | JSON | TOML | JSON |
| 配置文件位置 | OS 特定路径 | `~/.claude.json` 或项目 `.mcp.json` | `~/.codex/config.toml` | 内部管理 |
| 有 CLI 添加命令 | ❌ | ✅ | ✅ | ❌ |
| 有深链协议 | ⚠️ 新版 | ❌ | ❌ | ✅ |

所以 Pivot Web 最好能够**按客户端分开生成配置内容并复制到剪贴板**。

## 三、界面：Pivot Web 设置页

```
┌────────────────────────────────────────────────────────────┐
│  Pivot Web  >  设置  >  外部 AI 接入                        │
├────────────────────────────────────────────────────────────┤
│  🔑  访问令牌: pvt_xxx…xxx       [重新生成]  [撤销]         │
├────────────────────────────────────────────────────────────┤
│  选择你的 AI 客户端:                                         │
│   ┌──────────────────┐  ┌──────────────────┐               │
│   │  Claude Code     │  │  Cursor          │               │
│   │  [ 复制 Prompt ]  │  │  [ 一键安装 ]     │               │
│   └──────────────────┘  └──────────────────┘               │
│   ┌──────────────────┐  ┌──────────────────┐               │
│   │  Claude Desktop  │  │  Codex           │               │
│   │  [ 复制 JSON ]    │  │  [ 复制 CLI 命令 ]│               │
│   └──────────────────┘  └──────────────────┘               │
│   [ 其他 / 手动配置... ]                                    │
├────────────────────────────────────────────────────────────┤
│  已接入客户端: Claude Code (3 分钟前)                        │
└────────────────────────────────────────────────────────────┘
```

按客户端能力推荐不同方式：

| 客户端 | 推荐方式 | 理由 |
|---|---|---|
| Claude Code | 复制 Prompt 让 Claude 自己改配置 | Claude Code 自带文件读写，跨平台容错最强 |
| Cursor | `cursor://` 深链一键安装 | Cursor 原生支持 |
| Codex | 复制 CLI 命令 | 有 CLI 比手改 TOML 可靠 |
| Claude Desktop | 复制 JSON + 显示路径 | 无 CLI、深链不成熟，兜底方案 |

## 四、用户接入流程

           ┌─────────────────┬─┴──────────────┬──────────────┐
           ▼                 ▼                ▼              ▼

   ├──────────────┤  ├──────────────┤ ├──────────┤ ├──────────────┤

## 五、小结

MCP 服务端实施时**同步做这个设置页**，能把"第一次接入"从"踩坑 + 问人"变成"2 分钟自助搞定"。
