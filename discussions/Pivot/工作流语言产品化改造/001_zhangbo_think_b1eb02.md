# 工作流语言产品化改造

Pivot 当前仍直接暴露 think、act、verify、result、insight、quote、refer、status_change、paused -> executing、“同步 Git”等工程词，普通用户理解成本高。

需求：主视觉改中文：例如：think=想法/方案，act=执行。状态统一显示为讨论中、执行中、暂停中、已完成、已取消、已复盘、已经关闭。“同步 Git”改为“刷新数据/同步最新”。

请整体梳理项目中不符合非软件行业的同学能使用的词汇进行替换和映射
