通过 MCP 新建 Matter — 验收测试点                                                                                                                                           
                                                                                                                                                                              
  背景                                                                                                                                                                        
                                                                                                                                                                              
  外部 AI 终端（Claude / Cursor 等）通过 Pivot MCP 提供的 create_matter 工具，在 Pivot 中新开一条 Matter 并写入第一篇 timeline                                                
  文件。本次验收覆盖工具可见性、创建链路、落地数据、返回契约、错误体验五个面。

  范围

  - 工具：create_matter（首批 7 个 MCP 工具之一）
  - 入口：所有支持 MCP 的 AI 客户端
